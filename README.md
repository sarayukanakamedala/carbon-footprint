# 🌱 AI-Powered Carbon Footprint Intelligence System

An interactive **Streamlit web application** that estimates a user's carbon footprint based on lifestyle habits and provides personalized sustainability recommendations.

## Features
- 🌍 Carbon footprint estimation
- 🌿 Eco Score calculation
- 📊 Sustainability tier classification
- 💡 Personalized recommendations
- 🎨 Clean green-themed UI

## Technologies Used
- Python
- Streamlit
- Custom CSS
- Rule-based prediction model

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the app

streamlit run app.py

## Project Idea

This project analyzes lifestyle inputs such as:

- Fuel type
- Transport mode
- Food habits
- Electricity usage
- LPG consumption
- Travel frequency

It estimates monthly CO₂ emissions and provides actionable sustainability tips.
